## 🛠️ Overview
MonkeModManager simplifies how you manage mods for Gorilla Tag. This tool installs BepInEx and custom content for your game automatically. You do not need to move files or edit folders by hand. The manager finds your game files and handles the setup process. Use this tool to add cosmetics, custom maps, and various utilities to your VR experience.

## 💾 System Requirements
You need a Windows computer to run this application. Ensure you have the following installed before you start:
- Windows 10 or Windows 11.
- The latest version of Gorilla Tag installed through Steam.
- Sufficient space on your hard drive for game files. 
- A stable internet connection for the initial download.

## 🔧 Troubleshooting Common Errors
If you run into issues, try these steps first:

- The App Will Not Launch: Ensure you have the latest version of the .NET Desktop Runtime installed on your computer. You can find this on the official Microsoft website.
- Mods Do Not Work: Make sure you installed BepInEx through the manager. Start the game once after installing mods to let the files finish the setup process inside the game folder.
- Access Denied Error: Right-click the MonkeModManager icon and choose "Run as administrator." This gives the tool permission to change files in your game directory.
- Game Crashes: Some mods conflict with each other. If the game crashes, open the manager and uncheck the most recent mods you added. Restart the game to see if the issue goes away.
- File Path Not Found: Manually navigate your folders to ensure you selected the root folder for Gorilla Tag within SteamApps.

## 📝 Frequently Asked Questions
Will this tool break my game?
No. The manager creates a clear path for mods. If you want to remove your changes, you can simply uninstall the mods via the interface or verify your game files through Steam.

Is this tool safe?
This is a local tool. It only changes files inside your specific game folder. It does not send your data to outside servers.

Does this work with Quest versions?
This tool is for the PCVR version of the game on Steam. It does not support standalone Quest versions of the game. 

Can I play online with these mods?
Be careful when playing online. Some mod types might violate game rules. Use visual or aesthetic mods to stay safe. Avoid mods that change game physics or give you an unfair advantage over other players. 

How do I remove all mods?
Open the MonkeModManager. Click "Delete All Mods" or simply go to your game folder and delete the "BepInEx" folder. Then, use Steam to verify your game files to restore the original game state.
